⚠️ Note de sécurité

Pour une protection maximale, configurez les règles Firestore dans la console Firebase afin de restreindre la lecture des collections reports, adminConfig, users et moderation_logs aux seuls admins (champ isAdmin: true).

📜 Logs juridiques (LCEN / RGPD)

Chaque action de modération (suppression annonce, bannissement, traitement signalement…) est enregistrée dans la collection Firestore moderation_logs.

Collection Firestoremoderation_logs
Rétention minimale (LCEN)1 an
Rétention maximale (RGPD)3 ans
Champ TTL auto-suppressionexpiresAt

Activer la suppression automatique : Firebase Console → Firestore → Indexes → TTL → champ expiresAt sur moderation_logs.

Consultez et exportez les logs via l'onglet Logs juridiques du menu.
